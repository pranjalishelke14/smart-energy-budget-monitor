# Smart Energy Budget Monitoring System

A web + IoT based application that helps users monitor and control
their electricity usage by setting a monthly budget and tracking
appliance-wise consumption in real time.

## Features
- User authentication
- Appliance model-based power detection
- Monthly budget prediction
- Real-time energy monitoring using IoT
- Appliance-wise consumption analytics
- Alerts when usage exceeds daily limits

## Tech Stack
- Frontend: React.js
- Database: Firebase
- IoT: ESP32 + Current Sensor
- Visualization: Chart.js / Recharts
